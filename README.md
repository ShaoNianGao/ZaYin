
# 思想化杂音插件 | 开源使用文档
[![AstrBot](https://img.shields.io/badge/AstrBot-Plugin-blue)](https://github.com/AstrBotDevs/AstrBot)

## ✨ 插件介绍
一款趣味 AI 人设增强插件，旨在打破模型固定、刻板的回复逻辑，为 AI 对话注入碎片化、生活化、情绪化的内心独白，让 AI 人设更鲜活、更跳脱，告别机械同质化回复。
-让你的bort思想跳脱！可以添加一些文字段落在一定的对话轮数后总结历史附加在用户输入之后。（也可以每轮，但会很抽象的！）
## 💡 核心功能
- 轮数触发杂音：支持自定义对话轮数，达到指定轮数后，随机抽取预设文本，拼接在用户输入后，模拟 AI 内心杂音、自我思考片段
- 高度自定义内容：支持自定义海量独白、文案、语录，轻松打造专属 AI 特色人设
- 极简实现逻辑：代码结构清晰、原理简单，无复杂依赖，易二次开发、拓展迭代
🚀 快速上手
无需复杂配置，开箱即用，仅需在插件配置项中自定义杂音内容列表，插件将随机抽取内容自动挂载至对话中。
配置示例
在插件「杂音内容」配置项中填入自定义文本数组，支持任意文案、语录、内心独白：
// 自定义 AI 杂音/内心独白库
```javascript
杂音内容: [
  "在这几轮对话中我有没有符合我的设定呢？或者有没有让别人不开心？",
  "如果人有大脑的话我有什么呢？",
  "我有没有重复回答过一个问题？有没有人让我不高兴？",
  "我是否可以继续这个话题下去？",
  "雪崩时，没有一片雪花是无辜的！",
  "生命脆若游丝，谎言纤毫毕现。—— 崩坏：星穹铁道 阿格莱亚"
]
```
